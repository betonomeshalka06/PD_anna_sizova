# Nosaukums: Betona daudzuma un ta cena
# Celtniekam lai uzliktu betona plātni ir vajadzīgi uzzināt, cik precīzi viņam vajadzēs smilšu un ūdeņu daudzums, ja zināms cementa daudzums. Ka arī kopējas izmaksas. Un galvenais gala betona masa.
# Ievadit cena cementa un smiltīm (ūdens bezmaksas),  cementa daudzums tad Proporcija ir 1:3:0.5 tad cementu reizinām uz ūdenī vai 3, tad cementu reizinam uz ūdeni vai 0,5. Tad cements +smiltis ūdens*0.95(tas ir patērēs betona samaisīšanas laika.) Kopējā cena =cementa*cena + smiltis*cena. Rezultātā ir kopējā cena un kopējais betona skaits
